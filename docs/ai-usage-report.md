# AI Usage Report

## Recommended AI Tools Used
- **Google Gemini (AI Coding Assistant)**: Code generation, problem-solving, UI/UX improvements, and debugging.

## How the AI Tool Was Used
During the development of this portfolio assignment, the AI assistant was utilized in the following key areas:

1. **Dynamic Interactive Content**:
   - The AI provided recommendations and logic for creating a dynamic user greeting that calculates the current time of day (`new Date().getHours()`) and combines it with a personalized name field to output a custom welcome message.
   
2. **API Data Fetching**:
   - I used the AI to help write the asynchronous JavaScript logic (`fetchQuote`) to securely retrieve data from the *DummyJSON Random Quote API*. The AI suggested implementing a `try...catch` block to ensure the application fails gracefully and displays a friendly error message if the user is offline or the server fails.

3. **Client-Side Form Validation**:
   - The AI was used to assist in writing Javascript validation functions for the contact form, including verifying standard email formatting using Regex (`/^[^\s@]+@[^\s@]+\.[^\s@]+$/`) and updating the DOM to show clear `.error-message` spans when the inputs were invalid or empty.
   
4. **UI/UX Refinements**:
   - I utilized the AI to create full-page responsive layouts using Flexbox (`min-height: 100vh`) for the Quote section, ensuring that it matched the overall design aesthetic and carried over the required 'Dark Mode' variable support seamlessly.

## Learning Objectives & Understanding
Throughout this process, I carefully reviewed all code generated by the AI to ensure I understood the inner workings of `async/await` fetch requests, DOM manipulation (`getElementById`, `addEventListener`), and CSS overrides for dynamic states. The AI served as a powerful interactive tutor, teaching me modern JavaScript modularity and form validation standards without outright doing the assignment for me blindly.
